Este diretório contém uma skill master (orquestradora) e sub-skills especializadas para orientar um agente/IDE na criação de aplicações Spring Boot 4.x com boas práticas.

Use:
- skills/spring-boot-4x/SKILL.md → master
- skills/spring-boot-4x/<subskill>/SKILL.md → especializadas
- skills/spring-boot-4x/templates → templates auxiliares
